Алёша — десктопный мессенджер с клиент-серверной архитектурой. Основной функционал:

Авторизация по логину и паролю
Текстовые сообщения и отправка изображений
Личные чаты: переписка между двумя пользователями
Групповые чаты: общение нескольких участников
Реакции на сообщения
Аватарки у чатов/юзеров
Голосовые сообщения: запись, отправка, воспроизведение в чате
Безопасность: шифрование сообщений, безопасное хранение паролей, end-to-end шифрование

Антропович Пётр — GUI и клиентская часть

Реализация UI и пользовательского опыта (UX): верстка интерфейса на QML. Связывание UI с C++ кодом: обработка действий пользователя (клики, ввод текста) и обновление интерфейса в ответ на данные с сервера.
Управление локальными данными: кэширование данных (сообщения, профили) для ускорения загрузки. Работа с локальным хранилищем (настройки, сессия пользователя).
Стабильность desktop клиента: реализация жизненного цикла приложения (запуск, управление состоянием, корректное завершение). Обработка ошибок сети и исключений для предотвращения падений.
Тестирование: Unit-тестирование C++ модулей клиента.
